## 平台介绍：
**目前我们的蓝牙协议栈可以在STM32F103以及在Linux跑起来,蓝牙芯片是CSR8311,支持
<br>PROTOCOL：HCI L2CAP RFCOMM SDP
<br>PROFILE: HFP A2DP AVRCP PBAP SPP
